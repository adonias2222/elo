1. GLOSSÁRIO

I) Sistema Forponto – sistema desenvolvido pela empresa Task para preenchimento de dados, visualização do cartão de ponto, controle da jornada, registro de marcações de ponto e envio de eventos para folha de pagamento, afastamento médico, entre outros relacionados ao ponto do funcionário.

II) Forponto Mobile – é um aplicativo móbile componente do módulo web do sistema Forponto

III) Geolocalização – identificação ou estimativa da localização geográfica do usuário.

IV) Sistema operacional de dispositivo – realiza a execução de operações no aparelho.

V) Código ID (IMEI/UUID) - rótulo de identificação do dispositivo.

VI) Cache de dados – depósito onde os dados ficam armazenados na memória temporária do dispositivo.

VII) Cookies - identifica e armazenar informações do usuário que ficam gravados no dispositivo e podem ser recuperados pelo site que o enviou durante a navegação.

VIII) Links – ligação de uma palavra, texto ou imagem que quando é clicada pelo usuário, o encaminha para outra página na internet.

IX) Servidor de internet - estrutura onde armazena os dados e permitem o desenvolvimento e a execução do programa.

X) Software – programa que permite realizar tarefas específicas.

XI) Log – registro de operações.

XII) upload – o ato de transferir dados de um sistema de computador local para um servidor.

Este glossário serve apenas de referência e não limitará ou afetará o significado dos parágrafos ou itens aos quais se aplicam.
